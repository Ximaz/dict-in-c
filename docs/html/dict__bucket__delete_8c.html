int dict_bucket_delete ( bucket_t ** bucket,
char * key,
free_pair_t free_pair )

Deletes an entry from the bucket based on the key.

If no node matched the key, -1 is returned as an error and the bucket is unchanged.

Warning
If a NULL pointer is passed for either the bucket or the key, the function will crash.
Note
The key is not marked as const as it may get free'd by the free_pair function, but bear in mind the current function will not modify the key.
If the free_pair function is a NULL pointer, it's not called. It may be useful if neither the key nor the value were allocated.
Parameters
bucketThe bucket from which to remove the entry.
keyThe key used to match the entry to be removed.
free_pairThe function called to release the key and value memory.
Returns
0 on success, -1 on error.

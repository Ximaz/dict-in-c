Deletes an entry from the dict.

The function tries to delete an entry based on the given key. If the key was not found, the function returns -1 as an error. If the entry is found, it's key and value are given as parameter of the free_pair function to let the programmer handle the memory release, if needed.

Warning
If dict or key is a NULL pointer, the function will crash.
Note
The key is not marked as const as it may get free'd by the free_pair function, but bear in mind the current function will not modify the key.
If the free_pair function is a NULL pointer, it's not called. It may be useful if neither the key nor the value were allocated.
Parameters
dictThe dict from which the pair must be deleted.
keyThe key referring to the pair which must be deleted.
key_lengthThe length of the key. If unknowned, use strlen(key).
free_pairThe function called to release key and value memory.
Returns
0 on success, -1 on error.
